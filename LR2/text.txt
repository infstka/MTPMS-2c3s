типы бывают 
ссылочные и значимые

к ссылочным относят
string, dynamic, object
в переменных ссылочных типов хранятся ссылки на их данные

к значимым относят
целочисленные (byte, short, int, long)
с плавающей точкой (float, double)
bool, char
переменные значимого типа содержат в себе свои данные

разница между ссылочными и значимыми в том, что ссыл. хранит ссылку на данные, а знач. сами данные 

var
ключевое слово var может представлять любой тип, и какой это будет тип определяется во время компиляции

многомерный массив
многомерный массив индексируется двумя и более целыми числами

упаковка
когда любой значимый тип присваивается к ссылочному типу данных, значение перемещается из области стека в кучу. Эта операция называется упаковкой

локальная функция
локальная функция это функция внутри метода

отличие string от stringbuilder
string - неизменяемый, а stringbuilder - изменяемый, т.е. можно изменить/добавить/удалить символ 
при изменении string будет создана новая строка, содержащая старую с изменением 

equals
С помощью метода String.Equals можно легко определить идентичность двух строк. Этот метод учитывает регистр и возвращает логическое значение true или false. Метод можно вызывать из существующего класса, как показано в следующем примере. В примере ниже метод Equals используется для определения того, содержит ли строковый объект фразу "Hello World".

локальная функция
Локальная функция, как правило, содержит действия, которые применяются только в рамках ее метода.Локальная функция определяется как вложенный метод внутри содержащего ее элемента
